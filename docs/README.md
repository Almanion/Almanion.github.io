# Almanion.github.io — конспекты и материалы

Веб-сайт с материалами по предметам (в т.ч. 9 класс) и задачами МатЦентра.

## Разделы

- Конспекты по физике, алгебре, геометрии, химии
- Мини-проекты и отдельные страницы (например, «Мёртвые души» — каталог в `dead souls/`)

## Структура репозитория

- **Корень** — статика сайта: `index.html`, страницы предметов, `style.css`, `script.js`, папки `images/`, `dead souls/` и т.д. Для репозитория `username.github.io` GitHub Pages обычно отдаёт файлы из **корня** ветки `main`.

- **`docs/`** — вложенный репозиторий Git (подмодули; отдельная рабочая копия/история). Не смешивать с настройкой «папка `docs/` в Pages»: на GitHub сейчас публикуется **корень** этого репозитория.

- **`Local/`** — локальные заметки, скрипты и бэкапы; на сайт напрямую не рассчитано.

## Технологии

HTML, CSS, JavaScript; для формул — KaTeX там, где подключено на страницах.
