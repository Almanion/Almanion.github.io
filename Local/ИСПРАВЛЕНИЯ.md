### 1. 🔇 Выключенные анимации
**Проблема:** При выключении анимаций оставались некоторые hover-эффекты

**Решение:**
- Добавлено отключение всех трансформаций при наведении
- Отключены все box-shadow эффекты при hover
- **Исключение:** Изменение цвета фона для навигационных ссылок работает как и прежде

**Затронутые элементы:**
- Топики и карточки (`.topic`, `.task-card`, `.subject-card`)
- Блоки с информацией (`.definition-box`, `.formula-box`, `.remark-box`, и т.д.)

- Статистические карточки
- Telegram элементы
- Подсказки и модальные окна

**Код в `style.css`:**
```css
/* Отключаем все трансформации при наведении */
body.animations-off .topic:hover,
body.animations-off .task-card:hover,
/* ... и все другие элементы ... */
{
    transform: none !important;
    box-shadow: var(--shadow-sm) !important;
}

/* Разрешаем только изменение цвета фона для навигационных ссылок */

    transform: none !important;
    /* Цвет фона изменяется через стандартные стили */
}
```

### 2. 🎄 Баг с расширенными настройками снега
**Проблема:** При открытии расширенных настроек снега из меню настроек, их нельзя было закрыть

**Причина:** Конфликт имен функций `closeSettingsModal()` между `settings.js` и `newyear.js`

**Решение:**
1. **В `newyear.js`:**
   - **Переименованы все внутренние функции** для избежания конфликтов:
     - `openSettingsModal()` → `nyOpenSnowSettings()`
     - `closeSettingsModal()` → `nyCloseSnowSettings()`
     - `applySettings()` → `applySnowSettings()`
     - `resetSettings()` → `resetSnowSettings()`
   - Добавлен экспорт функций в глобальное пространство имен:
     ```javascript
     window.nyOpenSettingsModal = nyOpenSnowSettings;
     window.nyCloseSettingsModal = nyCloseSnowSettings;
     ```
   - Добавлен обработчик клавиши Escape для закрытия модального окна
   - Улучшена проверка: закрывается только если окно настроек снега открыто, а окно настроек сайта закрыто

2. **В `settings.js`:**
   - Использование глобальной функции `window.nyOpenSettingsModal()` для открытия настроек снега
   - Улучшена проверка Escape: закрывается только если окно настроек сайта открыто, а окно настроек снега закрыто

**Код в `settings.js`:**
```javascript
// Кнопка дополнительных настроек снега
nyAdvancedBtn.addEventListener('click', () => {
    closeSettingsModal(); // Закрываем наше окно настроек
    setTimeout(() => {
        // Используем глобальную функцию из newyear.js
        if (typeof window.nyOpenSettingsModal === 'function') {
            window.nyOpenSettingsModal();
        }
    }, 300);
});
```

**Код в `newyear.js`:**
```javascript
// Закрытие по нажатию Escape (только если это окно открыто)
document.addEventListener('keydown', (e) => {
    if (e.key === 'Escape') {
        const nyModal = document.getElementById('nySettingsModal');
        const settingsModalElement = document.getElementById('settingsModal');
        
        // Закрываем только если наше окно открыто и окно настроек сайта закрыто
        if (nyModal && 
            !nyModal.classList.contains('hidden') &&
            (!settingsModalElement || settingsModalElement.classList.contains('hidden'))) {
            closeSettingsModal();
        }
    }
});
```

1. Откройте меню настроек (⚙️)
2. Выберите уровень анимаций: **🔇 Выключено**
3. Наведите курсор на различные элементы:

   - ✅ По клавише Escape
