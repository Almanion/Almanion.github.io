# 🔓 Бессрочные Сессии

## ✨ Что изменилось?

Сессии теперь **не истекают**! 🎉

### Раньше:
- ❌ Сессия действовала 24 часа
- ❌ После истечения нужно было вводить пароль снова
- ❌ Неудобно при частом использовании

### Теперь:
- ✅ Сессия действует **бессрочно**
- ✅ Вход один раз → работаете вечно
- ✅ Пароль нужен только после **явного выхода**

---

## 🔐 Безопасность сохранена!

Не беспокойтесь о безопасности - все защитные механизмы остались:

### ✅ Что по-прежнему работает:

1. **Fingerprinting**
   - Сессия привязана к вашему устройству
   - Украденную сессию нельзя использовать на другом устройстве

2. **Шифрование**
   - Данные сессии зашифрованы
   - Без fingerprint их невозможно прочитать

3. **Блокировки**
   - 3 неудачные попытки → блокировка
   - Exponential backoff работает

4. **Мониторинг**
   - История попыток сохраняется
   - Детектор подозрительной активности активен

### 🛡️ Дополнительная защита:

**Сессия автоматически удаляется если:**
- Вы нажимаете кнопку "Выйти"
- Обнаружено несоответствие fingerprint
- Очищены данные браузера
- Вы явно вызываете `resetSecurityData()`

## 📋 Как это работает

### 1. Первый вход:
```
Вводите пароль →
Создаётся fingerprint устройства →
Генерируется сессия с expiresAt: Infinity →
Сессия шифруется и сохраняется
```

### 2. Последующие запуски:
```
Открываете страницу →
Система загружает сессию →
Проверяет fingerprint →
Если совпадает → автоматический вход ✅
Если не совпадает → сессия удаляется, запрос пароля ⚠️
```

### 3. Выход:
```
Нажимаете "Выйти" →
Сессия удаляется →
Все данные очищаются →
При следующем входе нужен пароль
```

## ⚙️ Технические детали

### Код (matcenter.js):

**Константа:**
```javascript
const SESSION_DURATION = Infinity; // Бессрочно
```

**Создание сессии:**
```javascript
const session = {
    token: generateSessionToken(),
    passwordHash: hashPassword(password),
    fingerprint: deviceFingerprint,
    createdAt: Date.now(),
    expiresAt: Infinity  // ← Бессрочно!
};
```

**Проверка при запуске:**
```javascript
if (session.expiresAt !== Infinity && session.expiresAt < Date.now()) {
    // Истекла (никогда не выполнится для бессрочных)
    clearSession();
}

if (session.fingerprint !== deviceFingerprint) {
    // Несоответствие устройства
    clearSession();  // ← Это сработает!
}
```

## 🔄 Как вернуть ограниченные сессии?

Если всё-таки хотите, чтобы сессии истекали:

### 1. Откройте `matcenter.js`

### 2. Найдите строку (около 16):
```javascript
const SESSION_DURATION = Infinity;
```

### 3. Замените на:
```javascript
const SESSION_DURATION = 24 * 60 * 60 * 1000; // 24 часа
```

**Другие варианты:**
- `7 * 24 * 60 * 60 * 1000` = неделя
- `30 * 24 * 60 * 60 * 1000` = месяц
- `365 * 24 * 60 * 60 * 1000` = год

### 4. Сохраните и перезагрузите страницу
