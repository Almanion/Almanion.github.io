# 💬 Telegram Коммуникация - Быстрый старт

## ✅ Что уже сделано:

1. ✅ **Создана серверная часть** (`Local/telegram-bot.gs`) - Google Apps Script для работы с Telegram Bot API
2. ✅ **Создана клиентская часть** (`telegram.js`) - JavaScript для фронтенда
3. ✅ **Добавлены стили** - красивый UI для диалогов и уведомлений
4. ✅ **Подключено ко всем страницам** - работает на всех HTML страницах сайта

## 🚀 Что нужно сделать (4 простых шага):

### Шаг 1: Развернуть Google Apps Script

1. Откройте https://script.google.com/
2. Создайте новый проект
3. Скопируйте код из `Local/telegram-bot.gs`
4. Нажмите **Развернуть** → **Новое развертывание** → **Веб-приложение**
5. **Важно**: Скопируйте URL (выглядит как `https://script.google.com/macros/s/XXXXX/exec`)

### Шаг 2: Создать Google Таблицу

1. Откройте https://sheets.google.com/
2. Создайте новую таблицу: **"Telegram Communications - Almanion"**
3. Скопируйте ID из URL (часть между `/d/` и `/edit`)
4. Откройте `Local/telegram-bot.gs`
5. Найдите `const SPREADSHEET_ID = 'YOUR_NEW_SPREADSHEET_ID';`
6. Замените на ваш ID
7. Сохраните и заново разверните скрипт (Шаг 1)

### Шаг 3: Обновить telegram.js

Откройте `telegram.js` и замените:

```javascript
apiUrl: 'https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec',
```

на ваш URL из Шага 1.

### Шаг 4: Запустить бота

Откройте Telegram и найдите вашего бота по имени или отправьте ему `/start`


**Способ 2: Через Google Таблицу**

1. Откройте вашу Google Таблицу
2. Перейдите на лист **Messages** (создастся автоматически)
3. Добавьте строку:

   - UserID: `user_abc123`

   - Sender: `admin`
   - Read: `false`

**Лист "Messages"** - все сообщения:

- Message

1. Проверьте URL в `telegram.js` (должен быть правильный)
2. Проверьте консоль браузера (F12) на ошибки
3. Проверьте, что бот запущен командой `/start`
4. Проверьте логи в Google Apps Script (View → Logs)
