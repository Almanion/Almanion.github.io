# 🤖 Инструкция по настройке Telegram коммуникации

## 📋 Описание системы

Система позволяет:
1. **Автоматически уведомлять** вас в Telegram, когда пользователь заходит на сайт
2. **Отправлять сообщения** пользователям через Telegram бота (они появляются как уведомления на сайте)
3. **Получать ответы** от пользователей обратно в Telegram

## 🚀 Установка и настройка

### Шаг 1: Настройка Google Apps Script

1. Откройте файл `Local/telegram-bot.gs`
2. Перейдите на https://script.google.com/
3. Создайте новый проект: **Файл** → **Создать** → **Скрипт**
4. Назовите проект "Telegram Bot для Almanion.github.io"
5. Скопируйте весь код из `telegram-bot.gs` в редактор
6. Сохраните (Ctrl+S или Cmd+S)

### Шаг 2: Деплой скрипта

1. В редакторе скриптов нажмите **Развернуть** → **Новое развертывание**
2. Выберите тип: **Веб-приложение**
3. Настройки:
   - **Описание**: "Telegram Bot API v1"
   - **Выполнять от имени**: Я
   - **У кого есть доступ**: Все
4. Нажмите **Развернуть**
5. **ВАЖНО**: Скопируйте URL развертывания (он выглядит так: `https://script.google.com/macros/s/XXXXX/exec`)

### Шаг 3: Настройка клиентской части

1. Откройте файл `telegram.js`
2. Найдите строку:
   ```javascript
   apiUrl: 'https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec',
   ```
3. Замените `YOUR_SCRIPT_ID` на скопированный URL из Шага 2
4. Сохраните файл

### Шаг 4: Создание Google Таблицы

⚠️ **ВАЖНО**: Создайте НОВУЮ Google Таблицу (НЕ используйте существующую!)

1. Откройте https://sheets.google.com/
2. Нажмите **+ Создать** → **Google Таблицы**
3. Назовите таблицу: **"Telegram Communications - Almanion"**
4. Скопируйте ID таблицы из URL:
   ```
   https://docs.google.com/spreadsheets/d/ВОТ_ЭТОТ_ID_НУЖЕН/edit
   ```
5. Откройте `Local/telegram-bot.gs` в редакторе
6. Найдите строку:
   ```javascript
   const SPREADSHEET_ID = 'YOUR_NEW_SPREADSHEET_ID';
   ```
7. Замените `YOUR_NEW_SPREADSHEET_ID` на скопированный ID
8. Сохраните и заново разверните скрипт

Автоматически создадутся два листа:
- **Visits** - история визитов пользователей
- **Messages** - история сообщений

### Шаг 5: Настройка Webhook (опционально, для двусторонней связи)

Для того чтобы вы могли отправлять сообщения пользователям прямо из Telegram:

1. В Google Apps Script редакторе найдите функцию `setWebhook()`
2. Замените `YOUR_SCRIPT_URL` на URL из Шага 2
3. Запустите функцию `setWebhook()`:
   - Выберите функцию `setWebhook` в выпадающем списке
   - Нажмите **Выполнить**
   - Дайте разрешения, если запросит
4. Проверьте логи (Ctrl+Enter) - должно появиться `"ok":true`

### Шаг 6: Проверка работы

1. Откройте ваш сайт в браузере
2. Проверьте консоль (F12) - должно появиться:
   ```
   ✅ Telegram коммуникация инициализирована
   👤 Ваш ID: user_XXXXX
   ```
3. В течение нескольких секунд вам должно прийти уведомление в Telegram

## 📱 Использование

### Как отправить сообщение пользователю через Telegram:

#### Метод 1: Через команды бота (требует Webhook)

1. Откройте Telegram и найдите своего бота (@YourBot)
2. Отправьте команду:
   ```
   /send user_XXXXX Привет! Как дела?
   ```
   где `user_XXXXX` - это ID пользователя из уведомления о визите

#### Метод 2: Через Google Таблицу (без Webhook)

1. Откройте Google Таблицу
2. Перейдите на лист **Messages**
3. Добавьте новую строку:
   - **Timestamp**: текущая дата и время
   - **UserID**: ID пользователя (например, `user_12345`)
   - **Message**: ваше сообщение
   - **Sender**: `admin`
   - **Read**: `false`
4. Пользователь получит сообщение при следующей проверке (каждые 15 секунд)

### Массовая рассылка всем активным пользователям:

Отправьте команду боту:
```
/broadcast Внимание! Обновление на сайте!
```

Сообщение получат все пользователи, которые были на сайте за последние 24 часа.

## 🛠️ Дополнительные команды бота

- `/start` или `/help` - показать справку
- `/send <userId> <message>` - отправить сообщение конкретному пользователю
- `/broadcast <message>` - отправить всем активным пользователям

## 🔐 Безопасность

- **User ID** генерируется локально и хранится в localStorage браузера
- **Сообщения** хранятся только в вашей Google Таблице
- **Доступ к боту** имеете только вы (по Telegram ID)
- **API токен** бота виден в коде, но бот настроен только на ваш ID

## 🐛 Решение проблем

### Не приходят уведомления в Telegram:

1. Проверьте, что токен бота правильный
2. Проверьте, что вы запустили бота командой `/start`
3. Проверьте логи в Google Apps Script (View → Logs)

### Пользователь не получает сообщения на сайте:

1. Проверьте URL Google Apps Script в `telegram.js`
2. Проверьте консоль браузера (F12) на наличие ошибок
3. Проверьте, что в Google Таблице есть непрочитанные сообщения для пользователя

### CORS ошибки:

- Используйте `mode: 'no-cors'` для POST запросов
- Для GET запросов используйте прямой URL с параметрами

## 📝 Дополнительные возможности

### Кастомизация уведомлений:

