# 🚀 Быстрый старт - Меню настроек

## 🎯 Что нового?

Теперь на вашем сайте есть **единое меню настроек** вместо отдельных кнопок! 

## 📍 Где найти?

Ищите кнопку **⚙️** в **правом верхнем углу** любой страницы сайта.

```
┌─────────────────────────────────────┐
│  📚 Конспекты            ⚙️ ← ТУТ! │
│                                     │
│                                     │
│        Ваш контент                  │
│                                     │
└─────────────────────────────────────┘
```

## 🎨 Что можно настроить?

### 1️⃣ Тема оформления
```
☀️ Светлая    |    🌙 Темная
   тема       |      тема
```

### 2️⃣ Новогодний вайб 🎄
```
❄️ ВЫКЛ  ━━━━O  ВКЛ ❄️

+ Кнопка: "🎨 Дополнительные настройки снега"
  (количество, скорость, размер, и т.д.)
```

### 3️⃣ Уровень анимаций
```
🚀 Максимальный  ← Все красиво, но медленнее
⚡ Средний       ← Баланс красоты и скорости  
🔇 Выключено    ← Быстро, но без эффектов
```

## 🎮 Как использовать?

### Шаг 1: Откройте меню
Нажмите на **⚙️** в правом верхнем углу

### Шаг 2: Выберите настройки
Кликните на нужную опцию - она применится **сразу**!

### Шаг 3: Закройте меню
- Нажмите **✕** в углу окна
- Или нажмите **Escape** на клавиатуре
- Или кликните вне окна

## 💡 Полезные советы

### Для слабых устройств:
1. Включите уровень анимаций **🔇 Выключено**
2. Уменьшите количество снежинок (если новогодний режим включен)

### Для комфортной работы ночью:
1. Включите **🌙 Темную тему**

### Для праздничного настроения:
1. Включите **🎄 Новогодний вайб**
2. Откройте дополнительные настройки снега
3. Настройте по вкусу!

## 🔄 Сброс настроек

Хотите вернуть всё как было?

1. Откройте меню настроек ⚙️
2. Прокрутите вниз
3. Нажмите **🔄 Сбросить всё**
4. Подтвердите действие

Страница перезагрузится с настройками по умолчанию.

## 📱 Работает везде!

✅ На компьютере  
✅ На планшете  
✅ На телефоне

## ❓ Проблемы?

### Кнопка ⚙️ не появляется?
- Очистите кеш браузера (Ctrl+F5)
- Перезагрузите страницу

### Настройки не сохраняются?
- Проверьте, что в браузере включены cookies
- Не используйте режим "Инкогнито"

### Снег лагает?
- Уменьшите количество снежинок в дополнительных настройках
- Или выключите новогодний режим
