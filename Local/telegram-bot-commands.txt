# Команды для настройки Telegram бота

## После создания бота через @BotFather:

1. Создайте бота:
   /newbot
   
2. Введите имя: 
   Almanion Site Bot
   
3. Введите username (должен заканчиваться на bot):
   almanion_site_bot

4. Сохраните токен который дал @BotFather:
   8567341752:AAGYi6tT0vOqtZsIRpK1m3Eoi9Izs2Q3CtQ
   (уже добавлен в telegram-bot.gs)

5. Настройте команды бота:
   /setcommands
   
   Затем отправьте:
   send - Отправить сообщение пользователю
   broadcast - Рассылка всем активным пользователям
   help - Показать справку

6. Настройте описание:
   /setdescription
   
   Бот для коммуникации с пользователями сайта Almanion.github.io
   
7. Настройте короткое описание:
   /setabouttext
   
   Система коммуникации для сайта с конспектами

## Примеры использования:

### Отправка сообщения конкретному пользователю:
/send user_abc123 Привет! Спасибо за визит!

### Массовая рассылка:
/broadcast Добавлены новые конспекты по физике!

### Справка:
/help

## Настройка webhook (для получения сообщений от бота):

1. В Google Apps Script откройте функцию setWebhook()
2. Замените YOUR_SCRIPT_URL на ваш URL
3. Выполните функцию
4. Проверьте логи - должно быть "ok":true

## Проверка работы:

1. Откройте сайт в браузере
2. В Telegram должно прийти уведомление о визите
3. Попробуйте отправить сообщение пользователю
4. На сайте должно появиться уведомление
