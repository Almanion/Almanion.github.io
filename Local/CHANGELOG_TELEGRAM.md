## 🔧 Технические детали:

### Архитектура:
```
Сайт (telegram.js)
    ↓ POST запрос
Google Apps Script (telegram-bot.gs)
    ↓ API запрос
Telegram Bot API
    ↓ уведомление
Ваш Telegram (@Almanion239)
```

### Проверка сообщений:
- Каждые 15 секунд клиент проверяет новые сообщения
- Используется long polling (через Google Apps Script)
- Сообщения помечаются как прочитанные после получения

### Хранение данных:
⚠️ **ОТДЕЛЬНАЯ** Google Таблица (вы создаёте её сами!)
- Лист **Visits**: история визитов
- Лист **Messages**: все сообщения

**НЕ ИСПОЛЬЗУЕТ таблицу МатЦентра!**

### ID пользователей:
- Генерируется автоматически при первом визите
- Формат: `user_XXXXX` (timestamp + random)
- Хранится в `localStorage`
- Анонимный (не содержит личных данных)

## 📊 Пример использования:

### Пользователь заходит на сайт:
1. Генерируется ID: `user_abc123`
2. Отправляется уведомление в Telegram:
   ```
   🔔 Новый посетитель на сайте!
   👤 ID: user_abc123
   📄 Страница: Физика
   ⏰ Время: 09.01.2025, 14:30:15
   ```

### Вы отправляете сообщение:
1. В Telegram боту: `/send user_abc123 Привет!`
2. Бот сохраняет в Google Таблицу
3. Через 15 сек пользователь получает уведомление на сайте
4. Открывает диалог и видит сообщение

### Пользователь отвечает:
1. Кликает на кнопку 💬
2. Пишет сообщение: "Спасибо за конспекты!"
3. Сообщение отправляется на сервер
4. Вам приходит в Telegram:
   ```
   💬 Сообщение от пользователя:
   👤 ID: user_abc123
   📝 Сообщение: Спасибо за конспекты!
   ⏰ Время: 09.01.2025, 14:35:20
   ```
