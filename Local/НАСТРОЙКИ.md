# 🎨 Меню настроек сайта

## 📋 Описание

Добавлено единое меню настроек для управления внешним видом и функциями сайта.

## ✨ Возможности

### 1. 🎨 Тема оформления
- **Светлая тема** ☀️ - классический светлый дизайн с высоким контрастом
- **Темная тема** 🌙 - комфортная темная тема для работы в условиях низкой освещенности

### 2. 🎄 Новогодний вайб
- **Включение/выключение** новогоднего режима со снегом и украшениями
- **Дополнительные настройки снега**:
  - Количество снежинок (10-200)
  - Скорость падения (0.2x-2x)
  - Размер снежинок (0.2x-2x)
  - Дрейф/покачивание (0-2x)
  - Непрозрачность (5%-100%)
  - Слияние снежинок при столкновении

### 3. ✨ Уровень анимаций
Три уровня производительности для комфортной работы на любых устройствах:

#### 🚀 Максимальный
- Все анимации и эффекты включены
- Плавные переходы и трансформации
- Декоративные анимации (shimmer, pulse, sparkle)
- Анимированные hover-эффекты
- Рекомендуется для мощных устройств

#### ⚡ Средний
- Основные анимации сохранены
- Упрощенные hover-эффекты
- Отключены декоративные анимации
- Ускоренные переходы (0.2s вместо 0.3s)
- Оптимально для большинства устройств

#### 🔇 Выключено
- Все анимации отключены
- Мгновенные переходы
- Нет hover-эффектов с трансформациями (масштабирование, вращение, и т.д.)
- Отключены все изменения box-shadow при наведении
- **Сохраняется:** изменение цвета фона навигационного меню при наведении
- Максимальная производительность
- Для слабых устройств и тех, кто предпочитает статический интерфейс

## 🎯 Как использовать

### Открытие меню настроек
1. Нажмите на кнопку **⚙️** в правом верхнем углу экрана
2. Откроется модальное окно с настройками

### Настройка темы
1. В разделе "🎨 Тема оформления" выберите одну из двух опций
2. Тема применится мгновенно
3. Выбор сохраняется автоматически

### Новогодний режим
1. В разделе "🎄 Новогодний вайб" переключите тумблер
2. Для точной настройки снега нажмите кнопку "🎨 Дополнительные настройки снега"
3. Настройте параметры снегопада по вкусу
4. Нажмите "✅ Применить"

### Управление анимациями
1. В разделе "✨ Уровень анимаций" выберите один из трех уровней
2. Изменения применятся мгновенно
3. Появится уведомление о текущем уровне

### Сброс настроек
1. Внизу меню нажмите кнопку "🔄 Сбросить всё"
2. Подтвердите действие
3. Все настройки вернутся к значениям по умолчанию
4. Страница автоматически перезагрузится

## 💾 Сохранение настроек

Все настройки автоматически сохраняются в localStorage браузера и применяются при следующем посещении сайта.

**Настройки по умолчанию:**
- Тема: Светлая
- Новогодний режим: Включен
- Уровень анимаций: Максимальный

## 📱 Адаптивность

Меню настроек полностью адаптивно и корректно работает на:
- 💻 Десктопах
- 📱 Планшетах
- 📱 Смартфонах

## 🔧 Технические детали

### Файлы
- `settings.js` - основная логика меню настроек
- `style.css` - стили для меню настроек и уровней анимаций
- Все HTML файлы обновлены для поддержки новой системы

### CSS классы для уровней анимаций
- `.animations-max` - максимальный уровень (по умолчанию)
- `.animations-medium` - средний уровень
- `.animations-off` - анимации выключены

### LocalStorage ключи
- `siteSettings` - основные настройки (тема, новогодний режим, уровень анимаций)
- `theme` - текущая тема (для совместимости)
- `newYearMode` - состояние новогоднего режима (для совместимости)
- `snowSettings` - настройки снегопада

## 🎉 Особенности

1. **Плавные переходы** между темами и уровнями анимаций
2. **Визуальный превью** тем перед применением
3. **Умные уведомления** о применении настроек
4. **Предупреждения** о производительности при большом количестве снежинок
5. **Автоматическая оптимизация** на мобильных устройствах
6. **Сохранение прокрутки** при переключении настроек

## 🐛 Известные ограничения

1. Новогодний режим может снижать производительность на слабых устройствах при большом количестве снежинок
2. Некоторые анимации могут работать по-разному в разных браузерах
3. При выключенных анимациях некоторые визуальные эффекты будут отсутствовать
