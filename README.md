# 📚 Almanion - Конспекты и Материалы

Веб-сайт с конспектами по различным предметам для 9 класса и задачами МатЦентра.

## 🎯 Разделы сайта

### 📖 Конспекты
- **Физика 9 класс** - Механика (кинематика, динамика)
- **Алгебра 9 класс** - Основные темы алгебры
- **Геометрия 9 класс** - Планиметрия и стереометрия
- **Химия 9 класс** - Конспекты по химии

## 🚀 Особенности

- ✨ Современный адаптивный дизайн
- 🌓 Светлая и темная темы
- 📱 Оптимизация для мобильных устройств
- 🔍 Поиск по конспектам
- 📖 Раскрывающиеся выводы формул
- ⚡ Быстрая навигация
- 🎨 Красивая типографика и формулы (KaTeX)

## 🛠️ Технологии

- HTML5
- CSS3 (с CSS переменными и анимациями)
- JavaScript (ES6+)
- KaTeX для математических формул
