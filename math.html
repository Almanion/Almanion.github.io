Функции и их свойства

1. Функция

Функция — это правило \(f\), сопоставляющее каждому элементу \(x\) из множества \(X\) единственный элемент \(y\) из множества \(Y\), обозначается \(f: X \to Y\).
💡 Обозначения:
• \(x\) — аргумент (независимая переменная)
• \(y\) — значение функции (зависимая переменная)
• \(X\) — область определения (множество допустимых значений \(x\))
• \(Y\) — область значений (множество всех значений \(f(x)\))
Область определения функции \(D(f)\) — множество всех значений аргумента, при которых функция имеет смысл.
Область значений функции \(E(f)\) — множество всех значений, которые принимает функция.
Отображение "на" или "суръекция" — это функция, которая отображает множество \(X\) на множество \(Y\) так, что во множестве \(Y\) нет "лишних" элементов \((Y = E(f))\).
Инъекция — это функция, которая отображает множество \(X\) на множество \(Y\) так, что у каждого элемента из множества \(X\) "свой" элемент из множества \(Y\) и они не повторяются.
Биекция — это функция, являющаяся одновременно инъекцией и суръекцией.
Образ \(x_0\) под действием \(f\) — это значение функции \(f(x_0)\).
Прообраз \(y_0\) под действием \(f\) — это элемент \(x_0\) из множества \(X\), такой что \(f(x_0) = y_0\). Обозначается \(f^{-1}(y_0)\).

2. Ограниченность функции

Функция \(f(x)\) называется ограниченной сверху на множестве \(X\), если существует число \(M\) такое, что для всех \(x \in X\) выполняется \(f(x) \leq M\).
Функция \(f(x)\) называется ограниченной снизу на множестве \(X\), если существует число \(m\) такое, что для всех \(x \in X\) выполняется \(f(x) \geq m\).
Функция \(f(x)\) называется ограниченной на множестве \(X\), если она ограничена и сверху, и снизу, то есть существует число \(C > 0\) такое, что \(|f(x)| \leq C\) для всех \(x \in X\).
Свойства ограниченных функций:
  • Если функция ограничена сверху и снизу, то она ограничена
  • Сумма ограниченных функций — ограниченная функция
  • Произведение ограниченных функций — ограниченная функция
  • Если \(|f(x)| \leq C\) на \(X\), то \(-C \leq f(x) \leq C\) для всех \(x \in X\)

3. Монотонная функция

Функция \(f(x)\) называется возрастающей на множестве \(X\), если для любых \(x_1, x_2 \in X\) таких, что \(x_1 < x_2\), выполняется \(f(x_1) < f(x_2)\).
Функция \(f(x)\) называется убывающей на множестве \(X\), если для любых \(x_1, x_2 \in X\) таких, что \(x_1 < x_2\), выполняется \(f(x_1) > f(x_2)\).
💡 Если неравенства нестрогие (\(f(x_1) \leq f(x_2)\) или \(f(x_1) \geq f(x_2)\)), то функция называется неубывающей или невозрастающей соответственно.
Свойства монотонных функций:
  1. Монотонная функция каждое своё значение принимает ровно один раз.
  2. Если функция \(f\) возрастает на \(X\), то \(-f\) убывает на \(X\).
  3. Если функции \(f\) и \(g\) возрастают на \(X\), то \(f + g\) также возрастает на \(X\).
  4. Если функции \(f\) и \(g\) возрастают на \(X\), \(k_1>0\), \(k_2>0\), то \(k_1f + k_2g\) также возрастает на \(X\).
  5. Если \(f\) возрастает и сохраняет знак на \(X\), то \(\frac{1}{f}\) убывает на \(X\).

4. Композиция функций

Композицией функций \(f\) и \(g\) называется функция \(h(x) = f(g(x))\), обозначается \((f \circ g)(x)\) или \(f(g(x))\).

Область определения композиции:

Для функции \(h(x) = f(g(x))\) область определения состоит из тех \(x\), для которых:

  1. \(x \in D(g)\) — \(x\) принадлежит области определения \(g\)
  2. \(g(x) \in D(f)\) — значение \(g(x)\) принадлежит области определения \(f\)

5. Монотонность композиции функций

Пусть функция \(f(x)\) монотонна на \(A\), \(g(x)\) монотонна на \(B\). Пусть функция \(\varphi(x) = g(f(x))\). Тогда:

  • Если \(f(x)\) возрастает на \(A\), \(g(x)\) возрастает на \(B\) , то \(\varphi(x)\) возрастает на \(A\);
  • Если \(f(x)\) убывает на \(A\), \(g(x)\) убывает на \(B\) , то \(\varphi(x)\) возрастает на \(A\);
  • Если \(f(x)\) убывает на \(A\), \(g(x)\) возрастает на \(B\) , то \(\varphi(x)\) убывает на \(A\).

6. Периодичность функции

Функция \(f(x)\) называется периодической с периодом \(T > 0\), если для всех \(x\) из области определения, \(x + T\) и \(x - T\) также принадлежат области определения, и выполняется: \[f(x + T) = f(x - T)\]
Основной период — наименьший период.

7. Дробно-линейная функция

Дробно-линейная функция — функция вида \[f(x) = \frac{ax + b}{cx + d}\] где \(a, b, c, d\) — действительные числа, причём \(c \neq 0\) и \(ad - bc \neq 0\).

Преобразование к стандартному виду:

\[f(x) = \frac{ax + b}{cx + d} = \frac{a}{c} + \frac{bc - ad}{c(cx + d)}\]

При \(bc - ad \neq 0\) это преобразуется к виду:

\[f(x) = A + \frac{B}{x - x_0}\] где \(A = \frac{a}{c}\), \(x_0 = -\frac{d}{c}\), \(B = \frac{bc - ad}{c^2}\).

9. Равносильные переходы в уравнениях

\[\sqrt{f(x)} = g(x) \Leftrightarrow \begin{cases} f(x) = g^2(x) \\ g(x) \geq 0 \end{cases}\] \[\sqrt{f(x)} = \sqrt{g(x)} \Leftrightarrow \begin{cases} f(x) = g(x) \\ f(x) \geq 0 \end{cases} \Leftrightarrow \begin{cases}] f(x) = g(x) \\ g(x) \geq 0 \end{cases}\]

10. Иррациональные неравенства

\[\sqrt{f(x)} > \sqrt{g(x)} \Leftrightarrow \begin{cases} f(x) > g(x) \\ g(x) \geq 0 \end{cases} \qquad \sqrt{f(x)} \geq \sqrt{g(x)} \Leftrightarrow \begin{cases} f(x) \geq g(x) \\ g(x) \geq 0 \end{cases}\]
\[ \sqrt{f(x)} > g(x) \Leftrightarrow \left[ \begin{array}{l} \begin{cases} f(x) \geq 0 \\ g(x) < 0 \end{cases} \\ \begin{cases} f(x) > g^2(x) \\ g(x) \geq 0 \end{cases} \end{array} \right. \qquad \sqrt{f(x)} \geq g(x) \Leftrightarrow \left[ \begin{array}{l} \begin{cases} f(x) \geq 0 \\ g(x) \leq 0 \end{cases} \\ \begin{cases} f(x) \geq g^2(x) \\ g(x) \geq 0 \end{cases} \end{array} \right. \]
\[\sqrt{f(x)} < g(x) \Leftrightarrow \begin{cases} f(x) < g^2(x) \\ f(x) \geq 0 \\ g(x) \geq 0 \end{cases} \qquad \sqrt{f(x)} \leq g(x) \Leftrightarrow \begin{cases} f(x) \leq g^2(x) \\ f(x) \geq 0 \\ g(x) \geq 0 \end{cases}\]
