Часть I: Векторная геометрия

1. Векторы / направленные отрезки

Вектор — упорядоченная пара точек.
Длина (модуль) вектора — длина соответствующего направленного отрезка.
\[|\vec{AB}| = AB\]
Направленные отрезки называются сонаправленными, если прямые, их содержащие, параллельны или совпадают, а относительно прямой, перпендикулярной им и проходящей через их концы, они лежат в одной полуплоскости.
Отношение эквивалентности — отношение, обладающее тремя свойствами:
  1. Рефлексивность: \(a \sim a\)
  2. Симметричность: \(a \sim b \Rightarrow b \sim a\)
  3. Транзитивность: \(a \sim b \text{ и } b \sim c \Rightarrow a \sim c\)
Вектор — класс эквивалентности направленных отрезков.
Свойства векторов:
  1. Если \(ABCD\) параллелограмм, то \(\vec{AB} = \vec{DC}\).
  2. Если \(\vec{AB} = \vec{CD}\), то \(\vec{AC} = \vec{BD}\).

2. Сложение и вычитание векторов

2.1 Сложение векторов

Правило треугольника: чтобы сложить векторы \(\vec{a}\) и \(\vec{b}\), нужно от конца вектора \(\vec{a}\) отложить вектор \(\vec{b}\). Тогда вектор, соединяющий начало \(\vec{a}\) с концом \(\vec{b}\), будет их суммой.
\[\vec{AB} + \vec{BC} = \vec{AC}\]
Правило параллелограмма: если векторы \(\vec{a}\) и \(\vec{b}\) приведены к общему началу, то их сумма изображается диагональю параллелограмма, построенного на этих векторах.
Применяется, когда векторы имеют общее начало
Свойства сложения векторов:
  1. \(\vec{a} + \vec{b} = \vec{b} + \vec{a}\)
  2. \((\vec{a} + \vec{b}) + \vec{c} = \vec{a} + (\vec{b} + \vec{c})\)
  3. \(\vec{a} + \vec{0} = \vec{a}\)
  4. \(\vec{a} + (-\vec{a}) = \vec{0}\)

2.2 Вычитание векторов

Разность векторов: вектор \(\vec{a} - \vec{b}\) определяется как сумма вектора \(\vec{a}\) и вектора, противоположного \(\vec{b}\).
\[\vec{a} - \vec{b} = \vec{a} + (-\vec{b})\]

3. Теорема о делении отрезка в заданном отношении

Теорема: Пусть точка \(M\) делит отрезок \(AB\) в отношении \(m:n\) (считая от точки \(A\)). Тогда \(\vec{OM}\) выражается через векторы \(\vec{OA}\) и \(\vec{OB}\) следующим образом:
\[\vec{OM} = \frac{n}{n+m}\vec{OA} + \frac{m}{n+m}\vec{OB}\]

По условию точка \(M\) делит отрезок \(AB\) в отношении \(m:n\), то есть:

\[\frac{AM}{MB} = \frac{m}{n}\]

Это означает, что:

\[\vec{AM} = \frac{m}{m+n}\vec{AB}\]

Выразим векторы через \(\vec{OA}\) и \(\vec{OB}\):

\[\begin{align*} \vec{OM} - \vec{OA} &= \frac{m}{m+n}(\vec{OB} - \vec{OA}) \\ \vec{OM} &= \vec{OA} + \frac{m}{m+n}\vec{OB} - \frac{m}{m+n}\vec{OA} \\ \vec{OM} &= \frac{(m+n)\vec{OA} + m\vec{OB} - m\vec{OA}}{m+n} \\ \vec{OM} &= \frac{n\vec{OA} + m\vec{OB}}{m+n} \end{align*}\]
Теорема: Пусть точки \(A\), \(B\), \(O\) не лежат на одной прямой. Тогда точка \(M\) принадлежит прямой \(AB\) тогда и только тогда, когда \(\vec{OM} = \alpha\vec{OA} + \beta\vec{OB}\), где \(\alpha + \beta = 1\).
Если \(\alpha \geq 0\), \(\beta \geq 0\), то точка \(M\) лежит на отрезке \(AB\).

4. Координаты вектора. Простейшие формулы в координатах

Линейной комбинацией векторов \(\vec{a}\) и \(\vec{b}\) называется вектор \(\alpha\vec{a} + \beta\vec{b}\).
\(\alpha\) и \(\beta\) называются коэффициентами линейной комбинации
Лемма: Если векторы \(\vec{a}\) и \(\vec{b}\) не коллинеарны и \(x\vec{a} + y\vec{b} = 0\), то \(x = y = 0\).
Теорема: Любой вектор можно представить единственным образом в виде линейной комбинации двух заданных неколлинеарных векторов.

\(\vec{a} = x\vec{b} + y\vec{a}\)

Пусть \(\vec{p}=x_1\vec{b}+y_1\vec{a}\)

\(\vec{0}=(x-x_1)\vec{b}+(y-y_1)\vec{a} \Rightarrow x-x_1=y-y_1=0 \Rightarrow x=x_1, y=y_1\)

Таким образом, вектор \(\vec{a}\) можно представить единственным образом в виде линейной комбинации двух заданных неколлинеарных векторов.

Базисом называется любая пара неколлинеарных векторов.
Свойства координат вектора:
  1. Два вектора равны тогда и только тогда, когда их координаты соответственно равны.
  2. При сложении векторов их соответствующие координаты складываются.
  3. При умножении вектора на число его координаты умножаются на это число.
Теорема: Если даны точки \(A(x_1; y_1)\) и \(B(x_2; y_2)\), то координаты вектора \(\vec{AB}\):
\[\vec{AB} = (x_2 - x_1; y_2 - y_1)\]
Теорема: Если даны точки \(A(x_1; y_1)\) и \(B(x_2; y_2)\), то длина вектора \(\vec{AB}\):
\[|\vec{a}| = \sqrt{a_x^2 + a_y^2}\]
Теорема: Расстояние между точками \(A(x_1; y_1)\) и \(B(x_2; y_2)\), вычисляется по формуле:
\[|AB| = \sqrt{(x_1-x_2)^2 + (y_1-y_2)^2}\]
Теорема: Если \(A(x_1; y_1)\) и \(B(x_2; y_2)\), то середина \(AB\) имеет координаты:
\[M\left(\frac{x_1 + x_2}{2}; \frac{y_1 + y_2}{2}\right)\]
Теорема: Векторы \(\vec{a} = (a_x; a_y)\) и \(\vec{b} = (b_x; b_y)\) коллинеарны тогда и только тогда, когда:
\[a_x b_y - a_y b_x = 0\]

5. Скалярное произведение векторов

Скалярное произведение векторов — число, равное произведению длин этих векторов на косинус угла между ними.
\[\vec{a} \cdot \vec{b} = |\vec{a}| \cdot |\vec{b}| \cdot \cos\varphi\]
Ненулевые векторы \(\vec{a}\) и \(\vec{b}\) перпендикулярны тогда и только тогда, когда их скалярное произведение равно нулю
Скалярное произведение в координатах: Если \(\vec{a} = (a_x; a_y)\) и \(\vec{b} = (b_x; b_y)\), то:
\[\vec{a} \cdot \vec{b} = a_x b_x + a_y b_y\]

6. Уравнения прямой

6.1 Общее уравнение прямой

Общее уравнение прямой — уравнение вида:
\[Ax + By + C = 0\]
где \(A\), \(B\), \(C\) — числа, причём \(A\) и \(B\) одновременно не равны нулю.
💡 Вектор \(\vec{n} = (A; B)\) называется нормальным вектором прямой (перпендикулярен прямой)

6.2 Уравнение прямой с угловым коэффициентом

Уравнение прямой с угловым коэффициентом:
\[y = kx + b\]
где \(k\) — угловой коэффициент (тангенс угла наклона прямой к оси \(Ox\)), \(b\) — смещение (ордината точки пересечения прямой с осью \(Oy\)).
\(k = \tan\alpha\), где \(\alpha\) — угол между прямой и положительным направлением оси \(Ox\)

6.3 Уравнение прямой, проходящей через две точки

Уравнение прямой через две точки: Если прямая проходит через точки \(A(x_1; y_1)\) и \(B(x_2; y_2)\), то её уравнение:
\[\frac{x - x_1}{x_2 - x_1} = \frac{y - y_1}{y_2 - y_1}\]

6.4 Уравнение прямой по точке и нормальному вектору

Уравнение прямой по точке и нормальному вектору: Если прямая проходит через точку \(M_0(x_0; y_0)\) и перпендикулярна вектору \(\vec{n} = (A; B)\), то:
\[A(x - x_0) + B(y - y_0) = 0\]

7. Расстояние от точки до прямой. Угол между прямыми

7.1 Расстояние от точки до прямой

Формула расстояния от точки до прямой: Расстояние от точки \(M_0(x_0; y_0)\) до прямой \(Ax + By + C = 0\) вычисляется по формуле:
\[\rho_l(M_0) = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}\]

Пусть прямая задана уравнением \(Ax + By + C = 0\), а точка имеет координаты \(M_0(x_0; y_0)\).

Вектор \(\vec{n} = (A; B)\) — нормальный вектор к прямой (перпендикулярен ей).

Возьмём произвольную точку \(M_1(x_1; y_1)\) на прямой. Тогда вектор \(\vec{M_1M_0} = (x_0 - x_1; y_0 - y_1)\).

Расстояние \(\rho_l(M_0)\) равно модулю проекции вектора \(\vec{M_1M_0}\) на направление нормали:

\[\begin{align*} \rho_l(M_0) &= \frac{|\vec{M_1M_0} \cdot \vec{n}|}{|\vec{n}|} \\ &= \frac{|A(x_0 - x_1) + B(y_0 - y_1)|}{\sqrt{A^2 + B^2}} \\ &= \frac{|Ax_0 + By_0 - (Ax_1 + By_1)|}{\sqrt{A^2 + B^2}} \end{align*}\]

Так как точка \(M_1\) лежит на прямой, то \(Ax_1 + By_1 = -C\), следовательно:

\[\rho_l(M_0) = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}\]

7.2 Угол между прямыми

Формула угла между прямыми (через угловые коэффициенты): Если две прямые заданы уравнениями \(y = k_1x + b_1\) и \(y = k_2x + b_2\), то угол \(\varphi\) между ними находится по формуле:
\[\tan\varphi = \left|\frac{k_1 - k_2}{1 + k_1k_2}\right|\]
При \(k_1k_2 = -1\) прямые перпендикулярны
Формула угла между прямыми (через нормальные векторы): Если прямые заданы общими уравнениями \(A_1x + B_1y + C_1 = 0\) и \(A_2x + B_2y + C_2 = 0\), то:
\[\cos\varphi = \frac{|A_1A_2 + B_1B_2|}{\sqrt{A_1^2 + B_1^2} \cdot \sqrt{A_2^2 + B_2^2}}\]

8. Эллипс

Эллипс — геометрическое место точек, для которых сумма расстояний до двух фиксированных точек (называемых фокусами) есть постоянная величина.
💡 Если \(F_1\) и \(F_2\) — фокусы эллипса, а \(M\) — произвольная точка эллипса, то \(|MF_1| + |MF_2| = 2a\), где \(a\) — большая полуось эллипса.

8.1 Каноническое уравнение эллипса

Каноническое уравнение эллипса с центром в начале координат:
\[\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1\]
где \(a > b > 0\), \(a\) — большая полуось, \(b\) — малая полуось.
💡 При \(a = b\) эллипс превращается в окружность

8.2 Основные элементы эллипса

Основные элементы эллипса:
  1. Фокусы \(F_1(-c; 0)\) и \(F_2(c; 0)\), где \(c = \sqrt{a^2 - b^2}\)
  2. Большая ось — отрезок длиной \(2a\), соединяющий вершины эллипса на большой оси
  3. Малая ось — отрезок длиной \(2b\), соединяющий вершины эллипса на малой оси
  4. Эксцентриситет \(e = \frac{c}{a}\), где \(0 < e < 1\) (чем ближе к 0, тем эллипс больше похож на окружность)
  5. Фокальное расстояние — расстояние \(2c\) между фокусами
Эксцентриситет эллипса — число \(e\), характеризующее степень "вытянутости" эллипса:
\[e = \frac{c}{a} = \frac{\sqrt{a^2 - b^2}}{a}\]
• При \(e \to 0\) эллипс приближается к окружности
• При \(e \to 1\) эллипс становится всё более вытянутым

8.3 Вывод уравнения эллипса

Пусть фокусы эллипса \(F_1(-c; 0)\) и \(F_2(c; 0)\), а точка \(M(x; y)\) лежит на эллипсе.

По определению эллипса:

\[|MF_1| + |MF_2| = 2a\]

Используя формулу расстояния между точками:

\[\sqrt{(x + c)^2 + y^2} + \sqrt{(x - c)^2 + y^2} = 2a\]

Перенесём второй корень в правую часть и возведём обе части в квадрат:

\[(x + c)^2 + y^2 = 4a^2 - 4a\sqrt{(x - c)^2 + y^2} + (x - c)^2 + y^2\]

Упростим:

\[4cx = 4a^2 - 4a\sqrt{(x - c)^2 + y^2}\] \[a\sqrt{(x - c)^2 + y^2} = a^2 - cx\]

Возведём в квадрат ещё раз:

\[a^2[(x - c)^2 + y^2] = a^4 - 2a^2cx + c^2x^2\] \[a^2x^2 - 2a^2cx + a^2c^2 + a^2y^2 = a^4 - 2a^2cx + c^2x^2\] \[a^2x^2 - c^2x^2 + a^2y^2 = a^4 - a^2c^2\] \[(a^2 - c^2)x^2 + a^2y^2 = a^2(a^2 - c^2)\]

Обозначим \(b^2 = a^2 - c^2\), тогда:

\[b^2x^2 + a^2y^2 = a^2b^2\]

Разделим обе части на \(a^2b^2\):

\[\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1\]

8.4 Свойства эллипса

Основные свойства эллипса:
  1. Эллипс симметричен относительно обеих координатных осей
  2. Эллипс симметричен относительно начала координат
  3. Эллипс ограничен прямоугольником со сторонами \(2a\) и \(2b\)
  4. Вершины эллипса: \(A_1(-a; 0)\), \(A_2(a; 0)\), \(B_1(0; -b)\), \(B_2(0; b)\)
  5. Площадь эллипса: \(S = \pi ab\)
  6. Оптическое свойство: луч света, выходящий из одного фокуса, после отражения от эллипса проходит через другой фокус

8.5 Уравнение эллипса со смещённым центром

Уравнение эллипса с центром в точке \((x_0; y_0)\):
\[\frac{(x - x_0)^2}{a^2} + \frac{(y - y_0)^2}{b^2} = 1\]
