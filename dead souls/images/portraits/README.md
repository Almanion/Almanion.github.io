# Портреты персонажей

Сюда поместите изображения персонажей. Имена файлов должны совпадать с `id` персонажа (формат PNG):

## Помещики (индивидуальные портреты)
- `manilov.png` — Манилов
- `korobochka.png` — Коробочка
- `nozdryov.png` — Ноздрёв
- `sobakevich.png` — Собакевич
- `plyushkin.png` — Плюшкин

## Главный герой
- `chichikov.png` — Чичиков

## Чиновники (общий обобщённый портрет)
Гоголь не даёт индивидуальных портретов чиновников города NN, поэтому для всех используется **один общий файл**:

- `official.png` — обобщённый образ чиновника

К этому файлу обращаются все 11 чиновников: губернатор, вице-губернатор, прокурор, председатель палаты, Иван Антонович «Кувшинное рыло», полицмейстер, почтмейстер, городской глава, инспектор врачебной управы, городской архитектор, управляющий казёнными фабриками.

---

**Рекомендации**: вертикальные изображения 400×500 или 600×800 px, формат PNG. Если файла нет — на странице будет показан стилизованный плейсхолдер с первой буквой имени персонажа.
